Mock-server is a Node.js web system for mock data.


## Features

 
  - Supports [Mockjs](http://mockjs.com/)
  - Supports ajax and jsonp
  - Supports custom domain

## Install dependencies

```
npm install
```
 - gulp 
 - gulp-connect
 - mockjs
## Start server

```
gulp
```
